RGB Values: a system that represents the colors in a computer display. Red, green, and blue are used to make any color in the visible spectrum.

Color Names: names of simple colors can also be used

Contrast: The difference between colors

CSS3 hsla: hsla stands for hue, saturation, lightness, and alpa, or opacity. This can be used to specify colors.

Hex Codes

A hex triplet is a six-digit, three-byte hexadecimal number used in HTML, CSS, SVG, and other computing applications to represent colors. The bytes represent the red, green and blue components of the color. One byte represents a number in the range 00 to FF (in hexadecimal notation), or 0 to 255 in decimal notation. The azure color of this paragraph is done with the hex code 50ccff

Hue

Hue is a degree on the color wheel (from 0 to 360) - 0 (or 360) is red, 120 is green,
240 is blue. Saturation is a percentage value; 0% means a shade of gray and 100%
is the full color. Lightness is also a percentage; 0% is black, 100% is white.
The green of this paragraph was done using a hue value

Brightness

Brightness is a filter you can apply to immages to set how bright they are, 0%
being pitch black, and 100% being normal brightness. This pineapple has 80%
of its normal brightness


Opacity

Opacity is the degree to which an element is transparent, 100%
being fully solid, 0% being fully transparent. This pineapple has
an opacity of 60%
